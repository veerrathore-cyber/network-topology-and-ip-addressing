# Network Topology and IP Addressing

## Project Overview
This project demonstrates the practical application of networking fundamentals by implementing a network topology consisting of two LANs connected through a router. The project focuses on understanding IP addressing, subnet masks, default gateways, and network communication concepts.

## Objectives
- Understand Public and Private IP addressing.
- Configure default gateways.
- Identify subnet masks.
- Analyze communication between multiple LANs.
- Strengthen networking fundamentals used in cybersecurity.

## Network Details

| Device | Private IP | Default Gateway | Subnet Mask |
|----------|------------|----------------|--------------|
| PC 1 | 192.168.1.2 | 192.168.1.1 | 255.255.255.0 |
| PC 2 | 192.168.1.3 | 192.168.1.1 | 255.255.255.0 |
| PC 3 | 192.168.2.2 | 192.168.2.1 | 255.255.255.0 |
| PC 4 | 192.168.2.3 | 192.168.2.1 | 255.255.255.0 |

## Concepts Covered
- Public vs Private IP Addresses
- Default Gateway
- Subnet Masks
- LAN Segmentation
- Router-Based Communication
